# Источники исторических событий

Файлы `*.tsv` в этой папке загружаются при старте приложения.

## Именование файлов

- **`*-2.tsv`** / **`*_2.tsv`** (например `2019-2.tsv`, `2018_2.tsv`) — менее важные события, показываются **только** на масштабах 30, 60 и 90 дней. На общем таймлайне (1y, 2y, 5y, 10y) скрыты, чтобы не перегружать.
- Обычные файлы (`2019.tsv` и т.п.) — показываются на всех масштабах.
